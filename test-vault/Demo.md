# Demo Note

Welcome to the **Subtitles MD** demo vault! This note shows you how to use the plugin.

## Transcript from External File

This transcript is loaded from a separate SRT file:

```transcript
file: Recordings/talk.srt
```

## Inline Transcript

You can also embed subtitle content directly:

```transcript
1
00:00:00,000 --> 00:00:02,500
This is an inline example.

2
00:00:03,000 --> 00:00:06,000
No external file needed!

3
00:00:06,500 --> 00:00:09,500
Perfect for quick notes and snippets.
```

## Try These Features

- **Click on subtitle files**: Open `Recordings/talk.srt` or `Recordings/lecture.vtt` to see the rendered view
- **Convert command**: Run "Convert subtitle file to transcript note" from the command palette (Ctrl+P)
- **Settings**: Configure timestamp display, paragraph breaks, and speaker formatting
- **Reading mode**: Switch to reading view (Ctrl+E) to see transcripts render beautifully
